🎧 WAVFORM — Music Player

WAVFORM is a sleek, modern, browser-based music player built using HTML, CSS, and JavaScript. It features a visually rich UI, authentication system, playlists, and a fully functional audio engine powered by the Web Audio API.

✨ Features
🔐 Authentication System
Sign In / Create Account
LocalStorage-based user management
Guest mode support
Persistent session (auto-login)

🎵 Music Player
Play / Pause / Next / Previous controls
Shuffle & Repeat modes
Real-time progress tracking
Volume control
Interactive waveform animation

📚 Library & Playlists
All Tracks view
Favorites system ❤️
Custom playlists:
Late Night 🌙
Energy Rush ⚡
Chill Waves 🌿

🎛️ Advanced UI
Dynamic “Now Playing” section
Animated waveform visualizer
Responsive sidebar navigation
Queue panel with "Up Next"
Sort tracks by:
Title
Artist
Duration

⚙️ How It Works
🔊 Audio Engine
Built using Web Audio API
Synthesizes sound using oscillators (sine, square, triangle, sawtooth)
Tracks are procedurally generated using:
Melody arrays
Bass patterns
BPM-based timing

💾 Data Storage
Uses localStorage for:
User accounts
Session persistence
🚀 Getting Started
Download or clone the project
Open index.html in your browser
Start using WAVFORM:
Create an account OR
Continue as Guest

⚠️ Limitations
No backend (all data stored locally)
Audio is synthesized (not real music files)
No file upload or streaming support
